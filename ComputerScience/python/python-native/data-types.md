# 🐍 Data Types

#### <mark style="color:yellow;">Casting</mark>

```python
# Int
new_int = int("0.1")

# String
new_string = str(0.1)

# Float
new_float = float("0.000220")
```

#### <mark style="color:yellow;">Check if a variable is a data type</mark>

```python
# Is it a string?
variable == str

# Is it an int?
variable == int

# Is it a float?
variable == float
```
